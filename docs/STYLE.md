# ПРИРОДНЫЙ МИНИМАЛИЗМ — официальный стиль дашборда

Источник: гайд заказчика. Это **единственная** палитра и типографика проекта.  
Дополнительно: **больше графиков, минимум текста**, **mobile-first**.

## Принципы

1. **Много воздуха** — крупные отступы между блоками (20–32 px).  
2. **Min text / max visualization** — абзацев нет; подпись к графику 1–3 слова.  
3. **Тёплые природные тона** — только палитра выше.  
4. **Каждый график = история** (готовность, комплектация, деньги, статусы задач).  
5. **MOBILE-FIRST** — ширина ~390–400 px, стек виджетов, tap ≥ 44 px.

---

## Виджеты (приоритет)

1. Scorecard в белой карточке (тень + border `#E8E0D8`)  
2. Gauge / horizontal bar / donut (2 сегмента)  
3. Компактная строка списка (точка + короткое имя + дата) — только если нельзя графиком  

Не использовать: плотные таблицы на 5+ колонок, длинные тексты рисков, фиолетовые/неоновые темы.
