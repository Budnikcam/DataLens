# Версии данных дашборда

Основное хранение версий теперь в браузере редактора (`localStorage`):

- `datalens-data-current` — активные данные;
- `datalens-data-versions` — массив снимков `{ id, savedAt, note, data }` (только добавление).

Папка `versions/` в репозитории — опциональный архив для разработчика (не обязательна для Маргариты).

| Файл | Роль |
|---|---|
| `../data.json` | Опубликованная версия на GitHub Pages (для всех посетителей) |
| `manifest.json` | Список файловых версий в репо (если писали через GitHub API) |
| `data-….json` | Снимок в репозитории |

## Как пользоваться (Маргарита)

1. Откройте https://budnikcam.github.io/DataLens/edit.html  
2. Войдите с паролем (см. `docs/HOW_IT_WORKS.md`).  
3. Правите поля → **«Сохранить»** (мгновенно, в этом браузере).  
4. Чтобы обновить сайт для всех — **«Скачать для публикации»** и отправьте `data.json` Ивану.
