# Версии данных дашборда

Каждое сохранение из редактора (`edit.html`) создаёт **новый** файл вида:

```
data-YYYY-MM-DD-HHmmss.json
```

Старые файлы **не удаляются** и **не перезаписываются**.

| Файл | Роль |
|---|---|
| `../data.json` | Текущая активная версия — её читает сайт на GitHub Pages |
| `manifest.json` | Список версий (только добавление записей) |
| `data-….json` | Снимок данных на момент сохранения |

## Как пользоваться

1. Откройте https://budnikcam.github.io/DataLens/edit.html  
2. Войдите с паролем (см. `docs/HOW_IT_WORKS.md`).  
3. Правите поля и нажмите **«Сохранить как новую версию (GitHub)»**  
   или **«Скачать версию»** (без GitHub).  
4. Через 1–2 минуты обновите главную страницу — подтянется новый `data.json`.

«Сделать текущей» в списке версий записывает выбранный снимок обратно в корневой `data.json` (старые файлы в `versions/` остаются).
